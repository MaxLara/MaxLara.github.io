Mecanizado CNC Silla Fresia

Objetivo General
  • Comprender el uso de corte en router CNC

Para este encargo realizamos el mecanizado de corte CNC para la silla “Fresia”. Debido a la actualidad, pudimos realizar simulaciones de corte y así poder ver como quedarían las piezas para el posterior armado de la silla.

Armado Partes Silla para corte

Para poder realizar los cortes CNC es necesario poder simular las planchas de donde se armaran las partes de la silla. A través del software de Fusion 360 podemos realizar dicha acción para posteriormente simular el corte de maquina. Cada parte de silla fue extraida de un modelo realizado en el mismo software.

Simulaciones cortes CNC

Una vez hecho el proceso anterior, se pueden llevar a cabo simulaciones de corte. Es necesario “setear” las opciones de corte como la fresa, cantidad y distancia de taps, profundidad de corte, entre otros.
